# Installation Commands Reference

## Install Backend Dependencies
npm install express@^4.18.2 mongoose@^7.5.0 cors@^2.8.5 dotenv@^16.3.1 firebase-admin@^11.10.1 cloudinary@^1.40.0 multer@^1.4.5-lts.1 express-validator@^7.0.1

## Install Backend Dev Dependencies
npm install --save-dev nodemon@^3.0.1

## Install Frontend Dependencies
npm install react@^18.2.0 react-dom@^18.2.0 react-router-dom@^6.15.0 react-scripts@5.0.1 axios@^1.5.0 firebase@^10.3.1 framer-motion@^10.16.1 react-icons@^4.11.0 react-player@^2.13.0 react-helmet@^6.1.0 react-lazy-load-image-component@^1.6.0

## Install Frontend Dev Dependencies (Tailwind CSS)
npm install --save-dev tailwindcss@^3.3.3 autoprefixer@^10.4.15 postcss@^8.4.29

## Initialize Tailwind CSS
npx tailwindcss init -p
